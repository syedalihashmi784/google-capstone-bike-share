Overview
This repository contains my **Google Data Analytics Capstone Project**, completed as part of the **Google Data Analytics Professional Certificate**.
The project analyzes bike-share usage data to identify trends and differences in rider behavior and to generate data-driven business insights.

---

Tools Used

  * Power Query
  * Pivot Tables and Charts
  
The analysis was performed primarily in Excel and Power Query, which is why this repository does not include programming code files.
